# @kortex/openrouter

**Status: Beta** — OpenRouter multi-model gateway adapter for Kortex.

| Capability | Supported |
|------------|-----------|
| Chat | Yes |
| Streaming | Yes |
| Embeddings | Yes |

## Environment

```env
AI_PROVIDER=openrouter
OPENROUTER_API_KEY=...
AI_MODEL=openai/gpt-4o-mini
```

Kortex is currently in alpha / developer preview.
