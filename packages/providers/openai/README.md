# @kortex/openai

**Status: Stable** — reference LLM adapter for Kortex.

| Capability | Supported |
| ---------- | --------- |
| Chat       | Yes       |
| Streaming  | Yes       |
| Embeddings | Yes       |

## Environment

```env
AI_PROVIDER=openai
OPENAI_API_KEY=sk-...
AI_MODEL=gpt-4o-mini
```

Kortex is currently in alpha / developer preview.
