# @kortex/gemini

**Status: Beta** — Google Gemini adapter for Kortex.

| Capability | Supported |
|------------|-----------|
| Chat | Yes |
| Streaming | Yes |
| Embeddings | Yes |

## Environment

```env
AI_PROVIDER=gemini
GEMINI_API_KEY=...
AI_MODEL=gemini-1.5-flash
```

Kortex is currently in alpha / developer preview.
